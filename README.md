# Test task at Teamrise

## Deploy
https://litelp.github.io/test-task/

## Requirements

Сверху: 
- логотип слева;
- кнопка «Войти» справа. 
По центру слева:
- заголовок на 2 строки;
- подзаголовок;
- ниже кнопка CTA (придумай текст сам, 1–2 слова, например: «Начать»).
Справа от текста — картинка/блок (можно просто цветной прямоугольник или SVG, без детали).

**Адаптив**
— на телефоне (320px): сначала текст, под ним кнопка, ниже блок с картинкой. 
— На компьютере (1366px и шире): текст и кнопка слева, картинка справа.

**Микро-эффекты** 
— При наведении на CTA кнопка должна чуть «подниматься» или давать тень. 
— Экран должен плавно появиться при загрузке (легкий fade-in).

**Правила** 
— Верстка чистая: HTML по смыслу (header, section, h1, p, button). 
— CSS отдельным файлом или в редакторе, без готовых шаблонов и копирования чужих секций. — Только один экран, без лишних блоков.
